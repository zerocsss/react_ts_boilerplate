# React boilerplate（工程化模板）

React + TS + Antd + Husky + Prettier + Docker + Travis

### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `yarn build`

#### husky+prettier 说明

#### docker + travis
持续集成，自动化docker部署
